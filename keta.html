Keta AI

"Private AI that lives on your hardware."

A local-first AI engine built from the ground up for performance and size. Run powerful language models on your own machine — no API keys, no data leaving your network, no subscription fees. Just fast, private inference.

System Requirements
OS: Linux · macOS · Windows
RAM: 4 GB minimum · 8 GB recommended
GPU: Optional — CUDA / Metal / CPU fallback
License: MIT — free, open, yours.
Features
  • ✓ Fully local — zero cloud calls, zero telemetry
  • ✓ Optimized for speed — quantized models, fast inference
  • ✓ Tiny footprint — models as small as 200 MB
  • ✓ Multi-provider — supports GGUF, ONNX, and more
  • ✓ Embeddable — use as a library or standalone server
  • ✓ Privacy by design — your prompts stay on your machine
  • ✓ Cross-platform — runs on Linux, macOS, and Windows

README.txt
AI should be yours.
Not someone else's API.
Not a monthly bill.

Keta AI is our take on what local AI should be: fast, small, and completely private. It runs inference directly on your hardware, supports multiple model formats, and is designed to be embedded into other K7I tools or used standalone. No API keys. No rate limits. No one watching.

Tech Stack
CoreRust — zero-cost abstractions
Inferencellama.cpp, candle, ONNX Runtime
ModelsGGUF, ONNX, SafeTensors
APIREST + WebSocket, OpenAI-compatible
BindingsPython, Node.js, Rust
UIOptional terminal + web interface
Architecture
keta/ ├── core/ inference engine ├── models/ model loader + registry ├── providers/ GGUF, ONNX, candle ├── server/ REST + WebSocket API ├── bindings/ Python, Node, Rust FFI ├── cli/ terminal interface └── lib.rs public API surface keta-ui/ ├── web/ optional web dashboard └── terminal/ TUI interface
